# Task 2: Exploratory Data Analysis (EDA)

## Overview

The objective of this task is to perform Exploratory Data Analysis (EDA) on an Amazon Product Reviews dataset. EDA helps in understanding the structure of the dataset, identifying patterns, checking data quality, and extracting meaningful insights before further analysis.

---

## Dataset Overview

- **Total Rows:** 2500
- **Total Columns:** 12
- **Date Range:** 2023-01-01 to 2024-07-04

### Columns

- Review_ID
- Product_Name
- Category
- Brand
- Price
- Rating
- Review_Text
- Review_Date
- Verified_Purchase
- Helpful_Votes
- Review_Length
- Discount_Percentage

---

## Data Inspection

The dataset contains:

- Numerical features
- Categorical features
- Boolean values
- Text data
- Date column

All columns have the correct data types and the dataset is ready for analysis.

The dataset was checked for missing values.

**Observation:**

- No missing values were found.

---

## Duplicate Records

The dataset was checked for duplicate records.

**Observation:**

- No duplicate records were found.

## Statistical Summary

Descriptive statistics were generated for all numerical columns.

**Key Observations**

- Product prices range from **₹533.83** to **₹49,991.15**
- Average product price is **₹25,023.89**
- Ratings vary between **1** and **5**
- Helpful votes and discount percentages show good variation across the dataset.

# Conclusion

The dataset is clean, complete, and suitable for further analysis.

Key findings include:

- The dataset contains **2500 customer reviews** across **35 products**, **7 categories**, and **10 brands**.
